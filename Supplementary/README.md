# TRI AI Saturdays Cohort 10 — Supplementary Teaching Materials

In partnership with **Google DeepMind**.

This directory contains supplementary resources for Sessions 1 and 2 of TRI AI Saturdays Cohort 10:
- **Session 1:** Intro to Language Models & BPE Tokenization
- **Session 2:** Static vs Contextual Embeddings

---

## Files

| File | Description |
|------|-------------|
| `bpe_cheatsheet.html` | "BPE Tokenization in 5 Steps" — beautiful, printable one-page cheat sheet with code snippets, a "cassava" walkthrough, and a complete Python BPE implementation. |
| `embeddings_summary.html` | "Static vs Contextual Embeddings" — two-column summary with Word2Vec vs BERT comparison, "Map of Meaning" metaphor, African context examples, and a detailed comparison table. |
| `quiz_questions.md` | 13–14 quiz questions across both sessions (multiple choice, polls, open-ended) with correct answers, explanations, and connections to African context and cohort projects. |
| `reading_list.md` | 24 curated readings organized by session, with skill-gap tags (`[#python]`, `[#ml-basics]`, `[#nlp]`, `[#advanced]`, `[#tools]`), URLs, and brief descriptions. |
| `setup_guide.md` | Step-by-step environment setup for Google Colab (recommended) and local Python. Includes a test script, troubleshooting guide, and quick-reference terminal commands. |
| `instructor_notes.md` | Detailed instructor notes with session timing, common student Q&A, debugging tips, mixed-skill classroom strategies, discussion prompts, and connections to career outcomes and project domains. |
| `README.md` | This file — overview of the directory contents and usage instructions. |

---

## How to Use the Materials

### Before Session 1
1. Share `setup_guide.md` with students at least 3 days before the session
2. Print or share `bpe_cheatsheet.html` as a session handout
3. Review `quiz_questions.md` and prepare the polling mechanism (Mentimeter, Google Forms, or in-person)
4. Review `instructor_notes.md` for timing estimates and common pitfalls

### Before Session 2
1. Print or share `embeddings_summary.html` as a session handout
2. Review Session 2 quiz questions and update your poll
3. Preview the Embedding Projector at https://projector.tensorflow.org/
4. Review African context examples in `embeddings_summary.html`

### Ongoing
- `reading_list.md` can be shared as a self-study resource
- `instructor_notes.md` should be annotated with your own observations after each session
